DataQuery

fetch remote data
insert_chartRange Query

Passing the chart spec's data parameter an object following the interface of { sheetId, range } with sheetId as a Google Sheet id and range as a Query Source Range, then a Query object will be created. The query will be sent automatically and update the chart with the data.

lava.chart({
  label: "Test",
  type: "PieChart",
  containerId: "chart_div",
  data: {
    sheetId: "1DwWSti6L3KRyJC0Wi33X3i-6lsZ2iStrhK7sxw7uoek",
    range: "A1:B6"
  },
  options: {
    width: 400,
    height: 240,
    is3D: true
  }
});

lava.draw();
